# music-explorer-webapplication


## Project Idea-

I am building a Music Explorer web application where users can search for songs and artists. The app will display song details like title, artist name, album image, and a short preview of the song.

## Purpose-

The purpose of this project is to practice JavaScript concepts like API integration, working with real-time data, and creating a responsive user interface.

## API Used-

I will use the iTunes Search API to fetch music data. It provides information about songs, artists, albums, and previews.

## Features-

* Search songs by name or artist
* Filter songs based on artist or type
* Sort songs alphabetically or by release date
* Play song preview
* Option to mark songs as favorites

## Technologies Used-

* HTML
* CSS
* JavaScript
* Fetch API

## How to Run the Project-

1. Clone the repository
2. Open the folder
3. Open index.html in a browser
